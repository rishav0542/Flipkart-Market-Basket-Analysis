# Flipkart Market Basket Analysis

**End-to-End Customer Insights & Product Recommendation System**

This project performs **Market Basket Analysis**, **Customer Segmentation**, and generates **actionable recommendations** for Flipkart using Python.

## 📌 Project Objective
To understand customer purchasing behaviour, identify frequently bought-together product categories, segment customers, and provide data-driven strategies to improve cross-selling, personalization, and customer retention.

## 🛠 Technologies Used
- Python
- Pandas, NumPy
- Seaborn, Matplotlib
- Scipy (K-Means)
- Itertools & Collections (Apriori)

## 📊 Key Insights from Analysis

### Market Basket Analysis (Apriori)
- **Clothing and Fashion** is the most central category with the strongest co-purchase relationships.
- Top association rules:
  - Clothing and Fashion + Beauty and Personal Care (Support: 0.278)
  - Clothing and Fashion + Home and Kitchen (Support: 0.266)
  - Clothing and Fashion + Groceries and Gourmet Food (Support: 0.261)

**Strongest bundle:** Clothing + Beauty products

### Customer Segmentation
Three clear segments were identified:
- **Frequent Buyers** (High frequency + High satisfaction)
- **Occasional Shoppers** (Medium frequency + Moderate satisfaction)
- **At-Risk Customers** (Low frequency/satisfaction + high cart abandonment)

### Satisfaction & Recommendation Insights
- Overall Shopping Satisfaction is moderate (~3.25/5).
- Personalized recommendations are only **somewhat helpful** for most customers.
- Customers trust real reviews more than Flipkart’s recommendations.
